import numpy as np
from typing import List

class DataAnalyzer:
  def __init__(self, data: List[float]):
    self.data = np.array(data)

  def analyze(self) -> dict:
    # Calculate basic statistics
    result = {
      'mean': np.mean(self.data),
      'max': np.max(self.data),
      'min': np.min(self.data)
    }
    return result

About EnCode

EnCode was created by developers for developers. We wanted an IDE that combines the speed of native applications with the flexibility of modern editors. Being developed in Rust gives EnCode unparalleled performance while the plugin system ensures it can grow with your needs.
